# Keremflix 🎬

A modern, browser-based local video player. Enjoy seamless video playback directly from your local filesystem with uncompromising privacy—no servers, no uploads, and your files never leave your machine.

[screenshot]

## ✨ Features

- **Secure Local Playback:** Utilizes the modern File System Access API to stream videos directly from your device. 
- **Smart History:** Securely remembers your recently played files, allowing you to jump back into your content instantly without having to manually locate the file again.
- **Dynamic Subtitles:** Fully responsive subtitle support that automatically scales perfectly with your video container.
- **Distraction-Free UI:** A clean, lightweight, and highly performant interface focused entirely on the viewing experience.

## 🛠️ Tech Stack

Built on the bleeding edge of the modern web ecosystem:

- **Next.js 16** (App Router)
- **React 19** & **TypeScript 6**
- **Tailwind CSS v4**
- **Zustand 5** (Client-side state management)

## 🚀 Quick Start

Get the player up and running locally in seconds.

### 1. Install dependencies
` ` `bash
npm install
` ` `

### 2. Start the development server
` ` `bash
npm run dev
` ` `

### 3. Open the app
Navigate to `http://localhost:3000` in your browser, select a local video file, and start watching!
